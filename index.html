Зачем это нужно

Почему VPN на роутере - лучшее решение для дома в 2026

Настроил один раз маршрутизатор - и все гаджеты в квартире обходят блокировки без лишних телодвижений

С августа 2024 Роскомнадзор через оборудование ТСПУ замедляет YouTube, полностью заблокировал Discord, Roblox, Instagram и Facebook. На компьютере или андроид-смартфоне можно поставить Zapret или ByeDPI и забыть. Но есть куча устройств, на которые вообще ничего не поставишь: телевизоры Samsung и LG, Apple TV, PlayStation 5, Xbox, умные колонки, детские планшеты.

Решение простое: настроить VPN или обход DPI прямо на Wi-Fi роутере. После этого любой девайс, который подключился к домашней сети, автоматически получает свободный интернет. Никаких программ, никаких настроек на каждом телефоне. Включил Wi-Fi - смотришь YouTube в 4K на телевизоре, звонишь в Discord на ноутбуке, играешь в Roblox на планшете.

Есть два подхода. Первый - VPN-туннель: трафик шифруется и идёт через зарубежный сервер (понадобится VPS в Европе — от 210 ₽/мес с оплатой рублями). Работает вообще с любыми заблокированными сайтами, включая Instagram. Второй - обход DPI (Zapret/nfqws): пакеты модифицируются прямо на роутере, без внешнего сервера. Быстрее, но не поможет с сайтами, заблокированными по IP (Meta). Можно и оба сразу.

Плюсы VPN на роутере

  • Работает для всех: телевизор, консоль, телефоны, планшеты
  • Настраиваешь один раз - работает постоянно
  • Не надо ставить приложения на каждый девайс
  • Гости автоматом получают доступ через Wi-Fi
  • Раздельное туннелирование экономит скорость
  • YouTube, Discord, Instagram, Roblox - всё работает

Минусы

  • Нужен совместимый роутер
  • Настройка посложнее, чем программа на ПК
  • VPN требует VPS-сервер (от 99 руб/мес)
  • Слабые роутеры могут просадить скорость
  • Не работает на мобильном интернете вне дома
Оптимальная схема: VPN на роутере с раздельным туннелированием для дома (ТВ, консоли, все Wi-Fi устройства) + VPN-приложение на телефоне (V2Box, ByeDPI) для мобильного 4G/5G на улице. Так вы прикрыты везде.

Keenetic

Настройка VPN на Keenetic - пошагово

Keenetic - самый популярный роутер для домашнего VPN в России. Поддерживает WireGuard, AmneziaWG и VLESS через Entware.

Способ 1: AmneziaWG (рекомендуем)

AmneziaWG - это модифицированный WireGuard, который ТСПУ не может заблокировать. Скорость до 500 Мбит/с на Keenetic Giga, хватит для 4K YouTube на нескольких телевизорах одновременно.

1 Арендуйте VPS

Нужен сервер за пределами России. Нидерланды, Германия или Финляндия - оптимально по пингу. Минимум: 1 ядро, 512 МБ RAM, Ubuntu 22.04. Цена от 99 руб/мес. Рейтинг хостингов: vps-dlya-vpn.vercel.app. Если ищете именно европейский сервер с оплатой в рублях — смотрите подборку VPS в Европе (10 провайдеров, от 210 ₽/мес). Бюджет ограничен? Есть рейтинг дешёвых VPS с тарифами от 99 ₽. Поставьте AmneziaVPN на комп, подключитесь к VPS, создайте AmneziaWG-конфиг. Скопируйте его - пригодится для роутера.

2 Обновите прошивку и компоненты

Зайдите в веб-интерфейс роутера: 192.168.1.1. Идите в Управление → Общие настройки → Компоненты. Проверьте, что стоят: «WireGuard VPN», «Подключение через SSTP», «Компоненты OPKG». Если WireGuard VPN нет - жмите «Изменить набор компонентов» и добавляйте. Обновите KeeneticOS до последней (5.0.4+).

3 Поставьте AmneziaWG через OPKG

Для AmneziaWG 2.0 нужен пакет amneziawg-go через OPKG Entware. Воткните USB-флешку в роутер (или используйте встроенную память на моделях с 256+ МБ). Подключитесь по SSH:

ssh admin@192.168.1.1

Поставьте Entware, затем AmneziaWG по инструкции из официальной документации Amnezia.

4 Импортируйте конфиг

В веб-интерфейсе Keenetic идите в Интернет → Другие подключения → WireGuard. Создайте новое подключение. Вставьте данные из конфига AmneziaWG: приватный ключ, публичный ключ сервера, Endpoint (IP:порт VPS), AllowedIPs. Хотите раздельное туннелирование? В AllowedIPs пропишите только подсети заблокированных сервисов, а не 0.0.0.0/0.

5 Проверьте

Включите соединение. Откройте YouTube на телевизоре или любом устройстве - видео должно грузиться мгновенно. Зайдите на whoer.net - если показывает IP вашего VPS, туннель работает.

Способ 2: VLESS Reality через Entware + Xray

Для максимальной устойчивости. VLESS маскируется под обычный HTTPS, ТСПУ вообще не может отличить ваш VPN от обычного серфинга. Нужен USB-накопитель и установка Xray через Entware.

1 Поставьте OPKG Entware

Воткните USB-флешку (ext4) в Keenetic. В веб-интерфейсе: Управление → Компоненты, поставьте «OPKG». Роутер закинет менеджер пакетов на флешку. Подробности: help.keenetic.com.

2 Поставьте Xray-core

Подключитесь по SSH. Ставьте зависимости и скачивайте Xray-core нужной архитектуры (mipsel для старых Keenetic, aarch64 для новых):

opkg update opkg install curl ca-certificates # Скачайте xray-core: github.com/XTLS/Xray-core/releases

3 Настройте VLESS Reality

На VPS поставьте 3X-UI (инструкция тут), создайте VLESS Reality inbound. Скопируйте конфиг и создайте /opt/etc/xray/config.json на роутере. Настройте маршрутизацию: только заблокированные домены через туннель.

4 Автозапуск

Создайте init-скрипт в /opt/etc/init.d/, чтобы Xray стартовал при загрузке роутера. Настройте iptables для перенаправления трафика. Подробнее: GitHub AlexFBG.

Что выбрать для Keenetic: AmneziaWG проще и быстрее, хватает для YouTube и Discord. VLESS Reality сложнее, но вообще не убиваемый - если завтра ТСПУ научатся фильтровать AmneziaWG, VLESS всё равно будет работать. Не уверены - начните с AmneziaWG.
OpenWrt

VPN на роутере с OpenWrt - Podkop и Passwall2

OpenWrt - свободная прошивка с полным контролем. Лучший вариант для VLESS, раздельного туннелирования и нестандартных задач.

Роутеры на OpenWrt (Xiaomi AX3000T, GL.iNet Flint 2, TP-Link Archer AX55, Cudy WR3000) дают максимум гибкости. Два главных инструмента для обхода блокировок:

Podkop - раздельная маршрутизация по доменам

Podkop (podkop.net) - open-source штука для OpenWrt, которая автоматом гонит трафик к заблокированным доменам через VPN, а остальной напрямую. Поддерживает VLESS, Trojan, Hysteria2, Shadowsocks. Встроенные списки заблокированных ресурсов.

1 Прошейте роутер на OpenWrt

Для Xiaomi AX3000T: качаете прошивку с openwrt.org, шьёте через веб-интерфейс стоковой прошивки или SSH. GL.iNet Flint 2 идёт с OpenWrt из коробки. TP-Link Archer AX55 - через TFTP Recovery.

2 Поставьте Podkop

Подключитесь по SSH и выполните:

opkg update opkg install podkop luci-app-podkop

Мало памяти? Соберите кастомную прошивку через OpenWrt Firmware Selector с включённым Podkop.

3 Настройте VPS и получите VLESS-ключ

На VPS ставите 3X-UI, создаёте VLESS Reality inbound. Копируете ссылку vless://.... В LuCI идёте в Services → Podkop. Вставляете VLESS-ссылку. Выбираете списки доменов для туннелирования: YouTube, Discord, Meta, Roblox.

4 Включайте и проверяйте

Жмёте «Enable» и «Save & Apply». Podkop сам настроит маршруты: заблокированные домены через VLESS, российские сервисы (Яндекс, ВК, Госуслуги) напрямую. Открываете YouTube на любом устройстве в сети - видео грузится моментально.

Passwall2 - продвинутая альтернатива

Passwall2 - мощный плагин для OpenWrt. Поддерживает VLESS, VLESS+XHTTP, Trojan, Hysteria2, Shadowsocks-2022. Есть графический интерфейс в LuCI, автообновление подписок, правила маршрутизации по доменам и GeoIP.

1 Установка

opkg update opkg install luci-app-passwall2 xray-core

Или соберите прошивку с Passwall2. После установки перезагрузите роутер.

2 Добавьте сервер

В LuCI: Services → PassWall 2 → Node List → Add. Тип - Xray/VLESS. Вбиваете данные VPS: адрес, порт, UUID, flow (xtls-rprx-vision), security (reality), SNI. Сохраняете.

3 Настройте маршрутизацию

Во вкладке Basic Settings: TCP Default - ваш VLESS-сервер, UDP Default - Direct (или тоже через сервер). Во вкладке Rule добавьте домены (youtube.com, discord.com, instagram.com). Остальной трафик пойдёт напрямую.

Podkop или Passwall2? Podkop проще, заточен под раздельное туннелирование, идеален если вы в этом деле новичок. Passwall2 мощнее, больше протоколов и настроек, для тех кто знает зачем ему это нужно. Оба бесплатные, оба open-source.
Asus / TP-Link

VPN на роутерах Asus и TP-Link

Стоковая прошивка Asus умеет WireGuard и OpenVPN, но в 2026 оба протокола блокируются. Выход есть.

Asus (AsusWRT / Merlin)

Asus из коробки поддерживает VPN-клиент: OpenVPN и WireGuard. На прошивке Merlin тоже есть WireGuard. Проблема: оба протокола режутся ТСПУ. Что делать:

Вариант А: VPN Fusion

На моделях с VPN Fusion (TUF-AX3000, RT-AX86U, RT-AX88U) можно выбрать, какие устройства ходят через VPN, а какие напрямую. Настройка: VPN → VPN Fusion → Add Profile. Загрузите .ovpn от коммерческого VPN с обфускацией (NordVPN Obfuscated, Surfshark NoBorders). Укажите устройства по MAC-адресу.

Вариант Б: Перешить на OpenWrt

Многие модели Asus (RT-AX58U, TUF-AX3000 V2) поддерживают OpenWrt. После перепрошивки получаете Podkop, Passwall2, VLESS Reality - всё то, что ТСПУ не блокирует. Проверяйте совместимость на openwrt.org.

TP-Link

Стоковая прошивка TP-Link почти не умеет в VPN-клиент (только OpenVPN на некоторых моделях, да и тот блокируется). Путь один - перешить на OpenWrt. Поддерживаются: Archer AX55, Archer AX53, Archer C6 v2/v3, Archer AX23. После установки ставите Podkop или Passwall2 по инструкции выше.

Xiaomi / Huawei / D-Link: Стоковые прошивки VPN-клиент не поддерживают. Единственный путь - OpenWrt (если модель совместима). Перед покупкой роутера обязательно проверяйте на openwrt.org/toh.
Split Tunneling

Раздельное туннелирование - только заблокированное через VPN

Зачем гнать весь трафик через VPN, если разблокировать надо только YouTube и Discord?

Раздельное туннелирование (split tunneling) - это когда через VPN идёт только трафик к заблокированным ресурсам (YouTube, Discord, Instagram, Roblox), а всё остальное напрямую к провайдеру. Что это даёт: Яндекс, ВК, Госуслуги, онлайн-банки работают на полной скорости без VPN; геолокация для российских сервисов не меняется; нагрузка на VPS минимальная; скорость практически не падает.

Как настроить

OpenWrt + Podkop

Podkop заточен именно под это. В настройках выбираете списки доменов (YouTube, Discord, Meta, Roblox) - только они пойдут через VLESS. Поддерживает автообновление списков из базы Antifilter.

OpenWrt + Passwall2

Во вкладке Rule добавляете домены и подсети. Поддерживает GeoIP и Geosite из баз Re:filter и Antifilter. Можно задать разные серверы для разных доменов.

Keenetic

В настройках WireGuard/AmneziaWG в AllowedIPs прописываете только подсети заблокированных сервисов вместо 0.0.0.0/0. Можно также использовать политики маршрутизации в «Приоритетах подключений».

Asus (VPN Fusion)

VPN Fusion позволяет выбрать конкретные устройства, которые ходят через VPN. Остальные напрямую. Разделение по устройствам, а не по доменам.

Списки заблокированных сайтов: Берите автообновляемые базы от сообщества: Antifilter (IP-адреса), Re:filter (домены и GeoIP для Xray/Sing-box). Podkop и Passwall2 умеют подтягивать их автоматически.
Без VPN-сервера

Zapret на роутере - обход DPI бесплатно и без VPS

Бесплатно разблокировать YouTube и Discord на всех устройствах. Внешний сервер не нужен - пакеты модифицируются прямо на маршрутизаторе.

Zapret (nfqws) - утилита для обхода DPI, которую можно поставить на Keenetic и OpenWrt. Она модифицирует TCP-пакеты так, что ТСПУ не понимает, что вы идёте на YouTube или Discord. Трафик идёт напрямую, без сервера - скорость максимальная, IP не меняется.

Плюс Zapret: полностью бесплатно, VPS не нужен, скорость как без VPN. Минус: не поможет с Instagram и Facebook (они заблокированы по IP, а Zapret IP не меняет).

Zapret на Keenetic

1 Поставьте OPKG Entware

Воткните USB-флешку в Keenetic. Поставьте OPKG в веб-интерфейсе (Управление → Компоненты). На моделях с 256+ МБ встроенной памяти (Giga, Hopper) Entware можно закинуть на встроенную память без USB: инструкция.

2 Поставьте Zapret

Зайдите по SSH и выполните:

opkg update opkg install curl ca-certificates iptables # Скачайте и запустите install_easy.sh из репозитория bol-van/zapret

Подробная инструкция для Keenetic: GitHub AlexFBG/zapret.

3 Проверьте

YouTube и Discord должны заработать на всех устройствах в сети. Не работает? Отредактируйте параметры nfqws в конфиге, подберите рабочую стратегию для вашего провайдера. На GitHub и 4PDA есть готовые конфиги для МТС, Билайн, МегаФон, Ростелеком.

Zapret на OpenWrt

1 Поставьте nfqws

opkg update opkg install nfqws # Или скомпилируйте из исходников: github.com/bol-van/zapret

2 Настройте правила iptables

Создайте скрипт в /etc/init.d/ с правилами для перенаправления трафика к YouTube и Discord через nfqws. Укажите стратегию обхода под вашего провайдера.

Zapret + VPN = максимум: Zapret для YouTube и Discord (бесплатно, без VPS, полная скорость), а VPN-туннель (VLESS/AmneziaWG) для Instagram, Facebook и всего, что заблокировано по IP. Оба метода отлично работают одновременно на одном роутере.

Безопасность

Безопасность и законность

Что надо знать про риски и правовой статус

Использование VPN для личных целей не является правонарушением в России. Законы (276-ФЗ, 90-ФЗ) направлены на операторов VPN-сервисов, которые не выполняют требования РКН, а не на обычных людей. Но правовая ситуация может измениться, так что следите за новостями.

По технической безопасности: со своим VPS вы полностью контролируете данные - никаких логов, никаких третьих лиц. Все инструменты (3X-UI, Zapret, Podkop, Passwall2) - open-source с открытым кодом на GitHub. Качайте только с официальных репозиториев.

Не покупайте «готовые VPN-роутеры» на Авито и Wildberries. В конце 2025 на Habr вышло расследование: такие устройства используют чужие WireGuard-конфиги, которые могут быть скомпрометированы или перестать работать когда угодно. Настройте сами - это безопаснее и надёжнее.
FAQ

Частые вопросы

Ответы на то, что спрашивают чаще всего

Как настроить VPN на роутере для YouTube?
На Keenetic: ставите AmneziaWG через OPKG, создаёте конфиг на VPS через Amnezia, импортируете в роутер. На OpenWrt: ставите Podkop, вставляете VLESS-ссылку с VPS. На Asus: VPN Fusion + обфусцированный провайдер. Подробно по шагам - в разделах выше.
Какой роутер взять для VPN?
Для простоты - Keenetic Giga или Hopper. Русский интерфейс, WireGuard из коробки, AmneziaWG через OPKG. Для максимума - Xiaomi AX3000T с OpenWrt: дёшево, мощно, свобода полная. Для премиума - GL.iNet Flint 2: OpenWrt из коробки, гигабайт оперативки, Wi-Fi 6.
VPN на роутере тормозит интернет?
С AmneziaWG или WireGuard просадка 5-30% на нормальных роутерах, для YouTube 4K за глаза. С OpenVPN - до 50-70% на слабых моделях. С раздельным туннелированием тормозит только заблокированное, Яндекс и ВК летают на полной скорости.
Можно бесплатно?
Zapret на роутере - полностью бесплатно, VPS не нужен. Обходит YouTube и Discord. Для Instagram нужен VPN-туннель, минимальный VPS от 99 руб/мес. В телеграм-каналах есть и бесплатные VLESS-ключи.
Работает ли для Smart TV?
Да, это главный плюс. Samsung, LG, Sony - любой телевизор на Wi-Fi автоматом получает доступ к YouTube, Netflix и остальному через стандартные приложения. На Android TV ещё SmartTube рекомендуем - YouTube без рекламы.
Как пустить только YouTube через VPN?
На OpenWrt: Podkop - он сам маршрутизирует по доменам. Добавляете youtube.com, googlevideo.com, discord.com - только они через туннель. На Keenetic: в AllowedIPs прописываете нужные подсети. На Asus: VPN Fusion выбираете устройства.
AmneziaWG или VLESS - что лучше?
AmneziaWG: проще настроить на Keenetic, очень быстрый, DPI не видит. VLESS Reality: максимально устойчив (маскируется под HTTPS), идеален для OpenWrt с Podkop. У МТС или Ростелеком с жёстким DPI - VLESS. У остальных - AmneziaWG проще и быстрее.
А для PlayStation 5 и Xbox?
Настраиваете VPN на роутере любым способом. Подключаете консоль к Wi-Fi - весь трафик автоматом через туннель. Roblox, магазины, мультиплеер - всё работает.
Мой роутер не поддерживает VPN. Что делать?
Три пути: 1) Перешить на OpenWrt, если модель подходит (проверяйте openwrt.org). 2) Купить второй роутер для VPN (Xiaomi AX3000T от 2500 руб) и подключить каскадом. 3) Запустить Zapret на компьютере и раздать по сети - но работает только пока комп включён.
WireGuard на роутере перестал работать
В 2025-2026 РКН начал блокировать обычный WireGuard через DPI. Переходите на AmneziaWG (модифицированный WireGuard, DPI не видит). На Keenetic ставите amneziawg-go через OPKG. На OpenWrt - меняете wg-клиент на AmneziaWG. Или переходите на VLESS Reality.
Что такое Podkop?
Open-source тулза для OpenWrt. Указываете домены (YouTube, Discord, Instagram) - только они идут через VPN. Остальное напрямую. Поддерживает VLESS, Trojan, Hysteria2. Ставится: opkg install podkop luci-app-podkop. Сайт: podkop.net.
Законно ли это?
VPN для личных целей - не правонарушение. Закон направлен на операторов VPN-сервисов, не на пользователей. Но ситуация может измениться. Свой VPN на VPS - наиболее безопасный вариант, вы ни от кого не зависите.

Словарь

Термины простым языком

VPN на роутере

Домашний Wi-Fi маршрутизатор с настроенным VPN-клиентом. Все устройства на Wi-Fi автоматом получают защищённое соединение и обход блокировок без дополнительных программ.

AmneziaWG

Модифицированный WireGuard от российского проекта Amnezia. Добавлена обфускация - ТСПУ не отличает AmneziaWG от обычного трафика и не может его заблокировать. Быстрый, низкие задержки.

VLESS Reality

VPN-протокол нового поколения. Маскирует трафик под обычный HTTPS к реальному сайту (например, microsoft.com). ТСПУ видит нормальный TLS-хендшейк и пропускает. Самый устойчивый к блокировкам на 2026 год.

Podkop

Тулза для OpenWrt, которая рулит раздельным туннелированием по доменам. Указываешь какие сайты через VPN, остальные напрямую. Поддерживает VLESS, Trojan, Hysteria2, Shadowsocks.

OPKG / Entware

Менеджер пакетов для установки доп. софта на Keenetic. Entware - репозиторий с тысячами пакетов. Через них ставится Zapret, Xray, AmneziaWG и прочее.

OpenWrt

Свободная ОС для роутеров на базе Linux. Заменяет стоковую прошивку и даёт полный контроль: VPN, файрвол, QoS, раздельное туннелирование. Поддерживается на сотнях моделей.

Раздельное туннелирование (Split Tunneling)

Через VPN идёт только часть трафика (YouTube, Discord), остальной напрямую. Сохраняется скорость для российских сервисов, снижается нагрузка на VPS.

ТСПУ

Оборудование DPI, установленное Роскомнадзором у провайдеров. Анализирует трафик и блокирует или замедляет запрещённые ресурсы. Именно ТСПУ тормозит YouTube и блокирует Discord.

3X-UI

Веб-панель управления для Xray-core. Ставится на VPS одной командой, даёт удобный графический интерфейс для создания подключений, генерации QR-кодов и мониторинга.

Zapret / nfqws

Open-source утилита для обхода DPI. Меняет структуру TCP-пакетов (фрагментация, десинхронизация) так, что ТСПУ не распознаёт запросы к YouTube и Discord. Работает без VPS - трафик идёт напрямую на полной скорости.
